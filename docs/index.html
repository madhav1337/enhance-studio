Enhance Studio

A privacy-first web app that batch-enhances photos, scans, newspaper clippings and book pages — drag, drop, download. Bulk & whole-folder upload.

▶ Try it live View on GitHub
Live demo on Render — first load may take ~30 s to wake

ℹ️ Enhance Studio runs its OpenCV image processing on the server, so the live demo is great for trying it out. For full privacy — images never leaving your machine — run it locally in under a minute.

Why it's nice

  • Bulk & folders — drop dozens of images, or a whole folder; a progress bar tracks the batch.
  • Before / after wipe slider — drag to compare original vs enhanced, pixel-for-pixel.
  • Four specialised pipelines — the right tool for photos, extractions, newspapers and book pages.
  • Download one, or all as a ZIP.
  • Non-destructive — originals are never touched.

Four pipelines

🪄

Photo Quality

upscale · denoise · sharpen

Cleaner, higher-resolution photos with composition, colours, faces and B&W preserved exactly — no crop, no AI invention.

🖼️

Photo Extract

crop · deskew · enhance

For phone photos of printed photos on a table: detects the print, perspective-corrects and crops it — never cutting the subject.

📰

Newspaper

natural archival restore

Auto-crop, deskew and text-edge sharpening so clippings read like a clean flatbed scan — paper texture & colour kept.

📖

Book / Document

cover or text page

Segments a cover onto a clean white canvas, or flat-field-whitens and sharpens a text page. Auto-picks the mode.

Tech stack

PythonFlaskOpenCV NumPyPillowgunicornDocker

A Flask server exposes a uniform enhance(pipeline, path, options) API over four pure image-processing engines. The front end handles the bulk queue, wipe-slider comparison and ZIP bundling — no build step.

Run it in a minute

git clone https://github.com/madhav1337/enhance-studio.git
cd enhance-studio
python -m venv .venv
# Windows: .venv\Scripts\activate
# macOS/Linux: source .venv/bin/activate
pip install -r requirements.txt
python app.py   # opens http://127.0.0.1:5000
